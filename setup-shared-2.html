Obtain a shared node API key

Now that you have created your identity and exported your encrypted private key, you only need a shared node operator to provide an API key to access their node. Join the idena.cyou telegram group to ask for an API key. For more options, join the idena shared nodes group to contact other node operators. Once you have your API key, or if you want to check your connection using a test API key, move to the last step of the tutorial: connect to the shared node.
