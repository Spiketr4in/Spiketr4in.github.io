Connect to the shared node

You are now ready to connect to the shared node. Go to the idena web app and follow the instructions below.

Idena web app Web app setup >

Sign in

Open the idena web app on your browser and click on Sign in.

Import your private key

At this point you will need to enter the encrypted private key you obtained previously and the password you used to generate it. Click Import.

Connect to Idena node

Here you will need to enter the URL of the shared node (for instance, https://node.idena.cyou to use our server) and the API key obtained from the respective shared node operator.

Activate your account

Congratulations, you connected to the shared node! At this point you have to use an invitation code to activate your account. You can obtain a code from another member of the Idena Network you might know or asking for it in the official Idena telegram group or in the Idena code sharing group.
